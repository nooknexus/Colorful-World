# Colorful World

**Instantly remove grayscale effects from any website and bring back the color with a single click!**

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/colorful-world/kjcgcfadnkhnehfldhdmdpjbbgpjhpeg)
- [Firefox Add-ons (AMO)](https://addons.mozilla.org/en-US/firefox/addon/colorfulworld/)

---

## Description

Are you tired of browsing websites that force a black-and-white or grayscale view? **Colorful World** is a simple yet powerful extension that gives you back control over your browsing experience. With just one click, you can disable all grayscale CSS filters on any webpage, instantly restoring its original colors.

Whether it's for accessibility reasons, design choices, or just a temporary site-wide effect, you no longer have to be stuck in a world without color.

### Features
*   **One-Click Toggle:** A simple and beautiful switch to turn the grayscale removal on or off.
*   **Works Everywhere:** Affects all open tabs and future pages you visit.
*   **Persistent Setting:** The extension remembers your choice. Set it once and browse freely until you decide to turn it off.
*   **Lightweight and Fast:** No unnecessary code, ensuring your browsing remains speedy.

### How to Use:
1.  Click the **Colorful World** icon in your browser's toolbar.
2.  A small popup will appear.
3.  Use the toggle switch to enable or disable the grayscale removal.
4.  Enjoy a more colorful web!

## Development

- The Chrome build lives at the project root using Manifest V3.
- The `firefox/` folder contains a Firefox-ready copy (Manifest V2) along with packaging notes in `firefox/README_FIREFOX.md`.
- Run `web-ext lint --source-dir=firefox` to validate compatibility before publishing updates to AMO.
